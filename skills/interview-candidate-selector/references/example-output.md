## 2. タイプAの定義と適合性評価

### タイプA：VPP主軸プレーヤー関係者の定義

> - **属性**: ネクストコネクト、アルファエナジー、その他VPPプラットフォーマーの事業企画・営業・経営企画部門の現職またはOB（経験3年以上）
> - **選定理由**: 契約交渉、価格設定、パートナー開拓の意思決定プロセスを直接経験しており、ビジネスモデルの裏側を知っている
> - **主に聞きたい論点**: ①②③（収益構造、契約実態、エコシステム構築）

## 【解説】この例のポイント

| 要素 | 設計意図 |
|------|----------|
| 候補者一覧の整理 | 全候補者を俯瞰し、カテゴリー・経歴・知見領域を一覧化 |
| タイプ定義の転記 | インタビューガイドの対象者タイプ定義を明示的に引用 |
| 適合性評価の可視化 | 経験の直接性と対応可能テーマの2軸で評価を可視化 |
| 推奨理由の具体性 | 論点との対応を明記し、なぜこの候補者を選ぶのかを説明 |
| 見送り理由の明確化 | 推奨候補との差異を説明し、見送りの妥当性を担保 |
| 質問フォーカスの設定 | 候補者ごとの強みを活かした質問設計を提示 |
